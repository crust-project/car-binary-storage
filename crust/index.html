Installation

pip install crust-shell

If using Linux, --break-system-packages may be necessary to make pip install properly. If you do not want to proceed with that, use pipx (not tested).

On Arch Linux, use AUR:

yay -S crust-git

Or with paru:

paru -S crust-git
